FTP Upload

FTP Upload Enable/disable upload of recorded events to a FTP service.
FTP Delete (after upload) Enable/disable delete of recorded events in camera storage after upload.
NOTE: If you enable this option, all copied files will be deleted from the cam.
FTP Host DNS or IP address of the FTP service.
FTP Directory Uploaded files will be placed into the specified directory. If it does not exist, it will be created. Leave empty to save files to the root directory of the FTP service.
Create directory tree Uploaded files will be placed into a directory tree. A tree equal to the one present in the cam will be made.
FTP Username Username to access the FTP service.
FTP Password Password to access the FTP service.
This change will take effect after the reboot of the camera.
